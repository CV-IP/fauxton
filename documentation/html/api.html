API Documentation¶

Overview¶

BlenderError An error that occurred executing code within Blender.
BlenderModule(source) Custom Blender functionality accessible via remote procedure calling.
BlenderResource A resource on the Blender server.
BlenderResourceType A type providing an interface to Blender resources.
enable_blender_gc() Allow the Blender server to automatically free unused resources.
disable_blender_gc() Prevent the Blender server from automatically freeing unused resources.
collect_blender_garbage() Manually free unused Blender resources.
Prop A graphical object that can be added to a scene.
Scene A collection of graphical objects.
read_scene(path) Read a scene from a ”.blend” file into memory.
write_scene(path, scene) Write a scene in memory to a ”.blend” file.
Camera A prop that can take snapshots of its surroundings.
DepthCamera A camera that reports the depth at each pixel.
SurfaceNormalCamera A camera that reports the surface normal at each pixel.
VelocityCamera A camera that reports the velocity at each pixel.

Cameras¶

class Camera¶

A prop that can take snapshots of its surroundings.

blender_type = 'Object:CAMERA'¶
render()¶

Take a snapshot.

look_at(target, roll=0)¶

Orient the camera towards a point in space.

field_of_view¶

The camera’s y and x viewing angles, in radians.

resolution¶

The camera’s y and x resolution, in pixels.

source¶

The OSL source to use as an emissive material when rendering.
