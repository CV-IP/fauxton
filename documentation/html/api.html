API Reference¶

Overview¶

BlenderModule Custom Blender functionality accessible via remote procedure calling.
BlenderError An error that occurred executing code within a BlenderModule.
BlenderResource A resource on the Blender server.
enable_blender_gc Allow the Blender server to automatically free unused resources.
disable_blender_gc Prevent the Blender server from automatically freeing unused resources.
collect_blender_garbage Manually free unused Blender resources.
Prop A graphical object that can be added to a Scene.
Action A keyframe-based animation that can be applied to a Prop.
Scene A collection of graphical objects.
read_scene Read a scene from a ”.blend” file into memory.
write_scene Write a scene in memory to a ”.blend” file.
Camera A prop that can take snapshots of its surroundings.
DepthSensor A camera that reports the depth at each pixel.
SurfaceNormalSensor A camera that reports the surface normal at each pixel.
VelocitySensor A camera that reports the velocity at each pixel.

class Scene(**properties)¶

Bases: fauxton.BlenderResource

A collection of graphical objects.

Parameters:**properties (dict) – Initial values of instance variables.
Operations defined on a Scene s:
len(s) Return the number of props in s.
iter(s) Return an iterator over the names of props in s.
n in s Return whether a prop is stored in s under the name n.
s[n] Return the prop stored in s under the name n.
s[n] = p Add the prop p to s, storing it under the name n.
del s[n] Remove the prop stored under the name n from s.

Cameras¶

class Camera(**properties)¶

Bases: fauxton.Prop

A prop that can take snapshots of its surroundings.

Parameters:

**properties (dict) – Initial values of instance variables.
Variables:
  • field_of_view (numpy.ndarray) – y and x viewing angles, in radians.
  • resolution (numpy.ndarray) – y and x resolution, in pixels.
  • source (str) – OSL source to use as an emissive material when rendering.
  • render_pass (str) – Blender render pass to use (e.g. “z” or “color”).
  • render_engine (str) – Blender render engine to use (e.g. “CYCLES”).
look_at(target, roll=0)¶

Orient the camera towards a point in space.

Parameters:
  • target (numpy.ndarray) – 3D spatial location to look at.
  • roll (float) – Rotation around the gaze axis, in radians.
